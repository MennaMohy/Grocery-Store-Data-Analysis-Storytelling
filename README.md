# Grocery Store Data Analysis & Storytelling (Tableau)

## 📌 Project Overview
This project presents an interactive data analysis of a grocery store dataset using Tableau Public. It focuses on exploring sales trends, product categories, and customer behavior through visual storytelling.

## 🛠️ Tools & Technologies
- Tableau Public

## 📊 Project Highlights
- Designed an interactive Tableau Story dashboard
- Analyzed sales distribution across product categories
- Explored customer purchasing behavior patterns
- Visualized key business insights for decision-making
